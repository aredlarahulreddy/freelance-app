\# Freelance Capital Finder



\## Project Description

Freelance Capital Finder is a Streamlit-based data analytics application that identifies ZIP codes with high freelance activity using IRS ZIP-level tax data.



The app helps in:

\- Detecting freelance-heavy regions

\- Segmenting ZIP codes into market categories

\- Visualizing economic activity (wages vs self-employment)

\- Mapping freelance hotspots across the United States

\- Providing AI-driven insights for marketing strategy

\## Features

\- Interactive filters for ZIP code analysis

\- Top ZIP code ranking based on freelance score

\- Market segmentation (Freelance Heavy / Balanced / Salary Heavy)

\- Scatter and bar chart visualizations

\- US map visualization of ZIP codes

\- AI Integration Strategy section

\- Sprint-to-Final roadmap
